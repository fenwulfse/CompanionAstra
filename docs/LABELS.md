# Issue Labels

This repo uses a small, consistent label set for triage and contributor work.

## Labels
- `bug` — Broken behavior or crashes
- `help wanted` — Tasks for contributors
- `ck-verify` — CK testing/validation
- `voice-mapping` — Player voice line mapping
- `docs` — Documentation updates
- `question` — Clarifications needed
- `good first issue` — Easy starter tasks
- `needs-triage` — New issues to review
- `blocked` — Waiting on external info
